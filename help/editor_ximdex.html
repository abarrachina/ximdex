4. El editor ximDEX

El editor ximDEX es una herramienta WYSIWYG (What You See Is What You Get -> lo que ves es lo que obtienes), gráfica e intuitiva que permite la creación de documentos XML.

El objetivo de esta herramienta es el de incluir y publicar documentos de forma que se pueda reducir el número de errores en la gestión de contenidos, además de popularizar el trabajo de publicación en Web debido a que su sencillez de uso permite que sea utilizada por un usuario con conocimientos básicos de informática.

Además, el editor ximDEX está integrado en una herramienta de control de publicación o workflow que gestiona de una forma integral todo el proceso de creación o eliminación de documentos dentro de un portal o sitio web a través de diferentes estados que ayudan a reducir y controlar los posibles errores en la actualización de contenidos.

El editor se compone de tres zonas o áreas:

Barra de archivo: donde se muestra el archivo editado

Barra de acciones: donde se ejecutan diferentes acciones con el documento

Zona de trabajo: donde se insertará el contenido del documento
