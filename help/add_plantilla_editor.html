3.4.3 Añadir plantilla de editor

Las plantillas de editor son aquellas que son visualizadas por el editor ximDEX y ayudan al usuario a formar los archivos XML. Dichas plantillas son necesarias en la creación de archivos XML con el editor ya que todos los archivos deben estar asociado a una. Las plantillas de editor están formadas por dos archivos:

- Plantilla de vista: permite visualizar el resultado final de las etiquetas XML

- Vista por defecto: contiene los datos iniciales o por defecto con los que debe comenzar un archivo en blanco asociado a dicha plantilla

Para insertar correctamente una Plantilla de editor, debe pulsar en el botón , dentro de la barra de acciones. A continuación, aparecerá en la zona de trabajo el siguiente formulario:

En este formulario, deberá introducir los archivos correspondientes a "Plantilla de vista" y "Vista por defecto" mediante el botón examinar. Una vez pulsado, debe elegir el archivo deseado, dentro de su equipo o red.

Una vez que se hayan seleccionado dichos archivos, deberá pulsar en "Subir Archivo" y la aplición le comunicará si el proceso de carga de dichos archivos ha finalizado correctamente.

Recomendación a la hora de añadir plantillas:

- Es aconsejable que el nombre de los archivos sea descriptivo al tipo de plantilla a cargar (p.e. si lo que vamos a visualizar son noticias, el nombre debería ser noticias-vista.xml y noticias.xml, respectivamente)
