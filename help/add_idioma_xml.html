3.8.4 Añadir un idioma a un documento XML

Para añadir o eliminar un idioma asociado a un contenedor XML deberá seleccionar el documento seleccionado dentro del árbol ximDEX, tal y como muestra la figura de la izquierda, después pulsar el botón dentro de la barra de acciones, lo que hará que aparezca el siguiente formulario dentro de la zona de trabajo:

Para "añadir un idioma", deberá seleccionar el "check" que hay situado en la columna "idioma" a la izquierda del idioma a añadir. Una vez pulsado, el sistema alertará de que el documento se creará cuando se pulse sobre el botón "modificar", situado en la parte inferior a la derecha del formulario, tal y como muestra la figura de la derecha.

Si desea "elimar un idioma" asociado a un documento XML, deberá quitar la selección del check que hay situado a la izquierda del idioma a eliminar. El sistema alertará en ese momento que el archivo se eliminará cuando pulse sobre el botón "modificar". Este cambio no se podrá deshacer y perderá todos los cambios realizados sobre dicho archivo y ximDEX no lo generará más.

Para completar cualquiera de las operaciones anteriormente descritas deberá pulsar sobre el botón "modificar", si no lo hace ninguna de las modificaciones realizadas tendrán efecto sobre los documentos.
