3.7.2.5 Cambiar el normbre a un archivo de la carpeta "css"

El usuario podrá cambiar en todo momento el nombre de las hojas de estilo contenidas en el proyecto. Para ello deberá seleccionar el archivo a borrar en el árbol ximDEX y después pulsar en el botón situado en la barra de acciones, lo que hará aparecer el siguiente formulario en la zona de trabajo:

En dicho formulario aparecerá el nombre actual del archivo seleccionado, una vez hecho esto deberá introducir el nuevo nombre de la imagen.

Advertencia: el cambio del nombre de una hoja de estilos en un proyecto web puede oscasionar que las páginas no se vean correctamente.
