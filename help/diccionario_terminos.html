Diccionario de términos ximDEX

- Archivo binario
- Archivo de texto
- Canal Un canal de publicación es una propiedad del contenedor de archivos XML y será el medio en el que se va a publicar un documento XML, como puede ser Web o PDA. Una vez asignada esta propiedad ximDEX se encargará automáticamente de generar el documento con las propiedades y formato del medio de publicaión del documento XML.
- Proyecto Un proyecto es un sitio web completo gestionado por ximDEX, formado por un conjunto de páginas XML, imágenes y hojas de estilo
- Servidor Zona del sitio web o portal que comprende una parte del proyecto y que comparte con éste plantillas y enlaces. Cada Proyecto debe contener, al menos, un servidor.
