3.7.2 La carpeta "css"

La carpeta "css" está destinada a almacenar las hojas de estilo del proyecto (archivos .css), que pueden utilizarse para todo el proyecto o sólamente en el nivel en el que están almacenadas.

Las hojas de estilo son las que controlan aspectos gráficos y tipográficos de un proyecto web, siempre que esté basado en ellas, con lo que la supresión, cambio de nombre o de ruta de estos archivos puede suponer la pérdida de la línea de diseño de todo o parte del proyecto.

En la carpeta "css" el usuario podrá:

- Añadir una carpeta "css"

- Añadir un archivo

- Editar archivos

- Descargar archivos

- Eliminar archivos
