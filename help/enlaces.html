4.2.3 Enlaces

Para insertar un enlace dentro de un documento XML deberá seleccionar primero el texto a enlazar, tal y como muestra la figura:

Una vez seleccionado, deberá pulsar sobre el icono en la barra de acciones del editor, lo que hará que aparezca la siguiente pantalla:

firgura 2.

Donde podrá seleccionar, a través del árbol ximDEX, el archivo a enlazar. Una vez seleccionado, el nombre de la página aparecerá en la caja de texto situada a la derecha de la pantalla.

Además podrá configurar las siguientes opciones para un enlace:

Clase: podrá seleccionar el estilo con el que desea que aparezca el enlace una vez publicado

Referenciador: con esta opción podrá elegir si desea que el enlace aparezca dentro de la misma ventana del navegador ("misma ventana") o en una ventana diferente ("ventana nueva")

Una vez seleccionadas las opciones deberá pulsar en "insertar enlace", para que los cambios surjan efecto dentro del documento. A partir de ese momento, los enlaces se identificarán mediante el icono, que se situará inmediatamente a la izquierda de la palabra o frase enlazada.

Editar un enlace

Para editar un enlace, deberá seleccionar la palabra o frases enlazados como muestra la siguiente figura

Cuando el texto esté seleccionado, deberá pulsar sobre el icono en la barra de acciones del editor, lo que hará que aparezca la ventana anterior (figura 2) sobre el editor, donde podrá modificar tanto el enlace, como el estilo o el referenciador. Una vez realizados los cambios deberá pulsar en "insertar enlace"

Eliminar un enlace

Para eliminar un enlace ya creado dentro de un documento, deberá seleccionarlo, tal y como se describe en párrafo anterior. Una vez que la pantalla de edición del enlace aparezca sobre el editor, deberá pulsar en el botón "eliminar enlace" y aceptar el mensaje de confirmación.
