3.7.1 La carpeta "common"

La carpeta "common", tiene como finalidad almacenar todos aquellos archivos que son usados por varias páginas a distintos niveles, como por ejemplo archivos "js" (archivos escritos en lenguaje JavaScript que ejecutan diversas funciones utilizadas por todo el sitio web). Dentro de esta carpeta, el usuario podrá:

- Añadir una carpeta "common"

- Añadir un archivo nuevo

- Descargar un archivo

- Eliminar archivos

- Cambiar el nombre a un archivo
