3.8 Documentos XML

Los documentos XML son los que necesitará crear un usuario para publicar información dentro de una sección o servidor. Un documento XMl consiste en un conjunto de entidades, que son principalmente texto y marcas. Con ximDEX, el usuario no tendrá que aprender ningún tipo de marcas propias del lenguaje, si no que será la herramienta, a través de su editor gráfico, la que se encargue de crear documentos XML que más tarde formarán las páginas HTML.

Los documentos XML únicamente se almacenarán dentro de la carpeta "_xml", donde el usuario podrá:

- Crear un documento XML

- Cambiar el nombre a un documento XML

- Añadir un idioma a un documento XML

- Eliminar un documento XML
