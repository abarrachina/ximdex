4.1 Conceptos básicos

Con el fin de guiar al usuario los archivos se mostrarán con el aspecto que tendrán una vez publicado y podrá introducir los elementos que corresponden a la plantilla seleccionada en el momento de la creación del archivo. Así, dentro de un archivo XML hay una serie de elementos fijos que conforman el entorno donde será publicado el archivo y los elementos editables y que serán los que podrá modificar el editor ximDEX.

Todos los elementos editables estarán marcados con un un icono identificativo, el cual cumple dos funciones: la de indicar al usario el tipo de objeto que es (párrafo, enlace, estilo, etc.) y la de dar acceso al usuario a las propiedades del objeto (url del enlace, estilo, etc.).

A continuación se describen las acciones básicas del editor, situadas en la Barra de Archivo y en la Barra de acciones de ximEDITOR:

Ruta: Situando el ratón encima de este botón se visualizará la ruta completa del archivo editado
Archivo: muestra el nombre del archivo editado en ximEDITOR.
Guardar: Guarda los cambios realizados en el documento editado.
Vista previa: Realiza una vista previa del documento en el canal elegido en el "combo" situado a la izquierda del botón.
Mostrar código XML: Muestra el código XML generado por el editor.
Deshacer: deshace la última acción realizada en el documento.
Rehacer: rehace la última acción deshecha en el documento.
Mover hacia abajo: Desplaza el elemento seleccionado una posición hacia abajo dentro de los elementos insertados dentro del documento editado
Mover hacia arriba: Desplaza el elemento seleccionado una posición hacia arriba dentro de los elementos insertados dentro del documento editado
