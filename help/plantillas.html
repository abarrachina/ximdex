3.4 Plantillas

Las plantillas son páginas XML formadas por código XML y HTML, que interpretan las páginas creadas a través del editor.

Estas plantillas están situdas en el raíz de cada proyecto y son comunes al mismo. Para explorar, añadir o eliminar dichas plantillas deberá abrir el proyecto deseado y, pulsar sobre el (+) situado a la izquierda la carpeta "_templates", tal y como muestra la figura de la izquierda.

En este punto usted podrá:
  - Añadir una nueva plantilla PTD
  - Añadir un archivo de texto ó
  - Añadir una nueva plantilla de editor
