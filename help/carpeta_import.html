3.7.3 La carpeta "_import"

La carpeta "_import" tiene como finalidad almacenar todos aquellos ficheros, de texto o binarios, que serán incluidos en algún documento XML, el uso más común de esta carpeta es el de almacenar páginas HTML que serán incluidas íntegras dentro de un documento XML para su posterior publicación dentro del portal.

Dentro de esta carpeta el usuario podrá:

- Añadir una carpeta "_import"

- Añadir un archivo nuevo

- Descargar un archivo

- Eliminar archivos

- Cambiar el nombre a un archivo
