1.4 Zona de trabajo

La zona de trabajo de ximDEX cumple la función de mostrar los documentos y formularios una vez que seleccione una acción dentro de la Barra de Acciones ximDEX.

Todas las acciones que ejecute el usuario se presentarán en esta zona y se abrirá una solapa por cada uno de ellas, de forma que se podrá visualizar cada una de las acciones abiertas pulsando sobre la solapa situada en la parte superior del documento. En el caso de que haya más solapas de las que se pueden mostrar en pantalla, el usuario podrá utilizar las flechas situadas en la parte superior derecha para moverse entre las mismas y mostrar las que estén ocultas.

Una vez finalizada la acción el usuario podrá cerrar cada una de las ventanas pulsando sobre la "x" situada a la derecha del nombre de la acción.
